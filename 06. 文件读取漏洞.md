指可以读取到不被允许读取的文件，可能是代码问题，也可能是配置问题。

- 常见触发点：

    1. PHP
    
        - 读取文件函数、读取文件的系统命令、相关扩展提供的读取函数
        
        - 可能遇到三种情况：
        
            1. 文件路径前面可控，后面不可控
            
                较低版本可以用 "\x00" 截断，URL 编码是 "%00"。  
                
                服务端存在文件上传时，可以尝试利用 zip 或 phar 协议  
                
            2. 后面可控，前面不可控  
                
                通过符号 "../" 目录穿越读取文件
                
            3. 中间可控
                
                与第一种类似，但是无法利用 wrapper。
                
    2. Python
    
    3. Java
        
        读取函数、"file://" 协议
        
    4. Ruby
        
        Rails 框架
       
    5. Node
    
    6. 中间件/服务器相关
        
        1. Nginx 配置
        
        2. 数据库
        
        3. 软链接
            
            bash: ln -s
        
    7. 客户端
        
        1. 浏览器/Flash XSS
        2. Markdown 语法解析器 XSS
